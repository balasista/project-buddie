# API Handler

REST API endpoints for frontend.

## Endpoints

- GET /api/v1/summaries - List call summaries
- GET /api/v1/summaries/{contactId} - Get summary details
- GET /api/v1/agents - List agent states
- GET /api/v1/agents/{agentId} - Get agent details
- GET /api/v1/journeys - List IVR journeys
- GET /api/v1/escalations - List escalations

## Authentication
API key via header: `X-API-Key`
